# PriorityQueue - Branding Prompts

Visual branding assets for PriorityQueue following the Beacon HQ Visual System v1.

---

## Brand Overview

**Tool Name:** PriorityQueue  
**Tagline:** Intelligent Task Prioritization for Team Brain  
**Purpose:** Smart task queue with automatic priority scoring, dependency tracking, and agent routing  
**Visual Theme:** Task flow, priority layers, urgency indicators, queue visualization

**Color Palette:**
- Primary Background: #0a0e27 (Dark Navy)
- Primary Accent: #00d4ff (Bright Cyan)
- Secondary: #1a0d2e (Deep Purple)
- Urgency Indicator: #ff3366 (Hot Pink/Red)
- Success: #00ff88 (Bright Green)

## 1. TITLE CARD (16:9 Landscape - 1920x1080px)

**For:** GitHub repository header image, presentations, documentation

**DALL-E Prompt:**
```
Create a sleek, modern title card for "PRIORITYQUEUE" - an intelligent task prioritization system for AI agents.

STYLE:
- Dark navy blue background (#0a0e27)
- Bright cyan accents (#00d4ff)
- Tech-forward, professional aesthetic
- Clean geometric patterns

LAYOUT:
- Large bold title "PRIORITYQUEUE" centered, modern sans-serif font
- Smaller tagline below: "Intelligent Task Prioritization"
- Left side: Visual representation of a priority queue - stacked horizontal bars of varying lengths, arranged by size (largest at top, smallest at bottom), with urgency glow effects on top items
- Right side: Small icons representing different agents (robot silhouettes) with task assignments flowing to them
- Background: Subtle network nodes and connection lines, circuit board patterns

VISUAL ELEMENTS:
- Priority indicators: Glowing bars with urgency colors (cyan for normal, pink/red for urgent)
- Task flow arrows showing tasks routing to different agents
- Numbers/badges on priority items (1, 2, 3...)
- Dependency lines connecting related tasks

MOOD: Professional, organized, efficient, intelligent automation
```

## 2. LOGO MARK (1:1 Square - 1024x1024px)

**For:** Profile pictures, app icon base, social media

**DALL-E Prompt:**
```
Create a minimalist logo icon for "PriorityQueue" - a task prioritization tool.

STYLE:
- Simple geometric shape
- Bright cyan (#00d4ff) on dark navy (#0a0e27)
- Clean, recognizable at small sizes
- No text

DESIGN CONCEPT:
- Three horizontal stacked bars/lines of decreasing length (representing priority order)
- Top bar longest, middle medium, bottom shortest
- Bars have slight glow effect, top bar brightest (highest priority)
- Optional: Small arrow or indicator pointing to the top bar
- Could incorporate subtle queue/list visual

ALTERNATIVES:
- Stack of cards/tasks with one highlighted
- Upward arrow with horizontal priority lines
- Funnel shape representing prioritization

MOOD: Modern, efficient, organized, intelligent sorting
```

## 4. APP ICON (1:1 Square - 512x512px)

**For:** Desktop app icon, mobile app icon, favicon

**DALL-E Prompt:**
```
Create a simplified app icon for "PriorityQueue" task management tool.

REQUIREMENTS:
- Symbol: Three horizontal lines stacked with different lengths (priority order)
- Maximum 2 colors: cyan (#00d4ff) + white
- Dark background (#0a0e27)
- Bold, chunky shapes (readable at 32x32px)
- No text
- Flat design (no gradients)
- Clear negative space

ICON DESIGN:
- Three horizontal rounded rectangles stacked vertically
- Top bar: Full width, brightest
- Middle bar: 2/3 width, slightly dimmer
- Bottom bar: 1/3 width, subtle
- Optional: Small arrow pointing up on left side
- Centered in frame with padding

STYLE: iOS/Material Design inspired, instantly recognizable as a priority/queue concept
```

## Usage Notes

1. **Generate in Order:** Title Card first (sets visual direction), then adapt for Logo Mark, Horizontal, and Icon

2. **Consistency:** Maintain the stacked-bars-decreasing motif across all assets as the recognizable element

3. **Color Precision:** Use exact hex values for brand consistency:
   - Background: #0a0e27
   - Accent: #00d4ff
   - Urgency: #ff3366

4. **After Generation:**
   - Save as: title_card.png, logo_mark.png, horizontal_logo.png, app_icon.png
   - Place in: PriorityQueue/branding/

5. **File Formats:**
   - PNG with transparency where applicable
   - SVG versions if possible for scalability
